# **VeinManager**
	矿物控制：对矿物自动整理.
	《全矿物添加》模式，《全矿物添加》模式后，添加全部矿物，每样矿物1000×10×10000份
	因为一个星球可能仅用一次，快捷键设置：Ctrl+Alt+Shift+V
