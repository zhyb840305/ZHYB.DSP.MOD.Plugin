# SuperAccumulator
	超级蓄电池，1E的电量，足以用到游戏结束；
	总电量：	E 1000(P)*1000(T)*1000(G)*1000(M)*1000(K)*1000；
	最大输入输出：		T 1000(G)*1000(M)*1000(K)*1000;
	假如用完了，拆掉再放一个就行了；
	拆除进背包的是满电。

------------


# Installation
	1.Install BepInEx
	2.Then drag .dll into BepInEx/plugins
